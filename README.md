# Blog API (Author App)

## Blog API (Author App) - Overview

This is the sixth project of the "NodeJS" course within "The Odin Project".

This project is used to practice the implementation of a RESTful application.

In particular, this project aims to create a blog platform.

**Note:** This repository is for the author application.
This side of the application only includes the functionality needed to create, edit and delete blog posts.

For the reader application visit [this repository](https://github.com/Dur4nt3/blog_project_reader).

For the api backend visit [this repository](https://github.com/Dur4nt3/blog_project_api).
